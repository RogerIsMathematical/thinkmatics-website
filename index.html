Today's Challenge

Problem 1: The Consecutive Odd Numbers

Can you find three consecutive odd numbers that sum to 27?

Hint: If the first odd number is n, what would the next two consecutive odd numbers be? (Remember, odd numbers are 2 apart!)

Problem 2: Pattern Discovery

Look at this sequence: 1, 4, 9, 16, 25, ... What's the pattern? What's the 10th number?

Hint: Try looking at the differences between consecutive numbers. Or think about what these numbers represent geometrically...

Problem 3: The Handshake Problem

If there are 5 people in a room and everyone shakes hands with everyone else exactly once, how many handshakes occur?

Hint: Start small. What if there were only 2 people? 3 people? Can you find a pattern?
